### Repository Structure

#### [Binary Exploitation (binex)](https://stack.int.mov/rop-chain-exploits/)


Focused on the "Really Secure Notes" challenge. This project demonstrates a **CET-compliant Vtable Hijack** against an Ubuntu 26.04 environment hardened with Intel CET (SHSTK/IBT), Full RELRO, and PIE. It proves that application-level logic remain vulnerable even when hardware protections are at their peak.

#### [Web Exploitation (webex)](https://www.google.com/search?q=./webex)

Focused on advanced web application vulnerabilities. This section includes documentation, source code analysis, and proof-of-concept exploits for modern web architectures, detailing the bypass of common web-tier defenses.
